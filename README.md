# 🧬 Generative AI Drug Discovery

## 📌 Project Overview

Generative AI Drug Discovery is an AI-powered healthcare application designed to generate and analyze drug-like molecules using Generative AI concepts, SMILES molecular representation, and computational chemistry techniques.

The system simulates early-stage computational drug discovery by generating molecular structures, validating them, predicting pharmaceutical properties, performing toxicity analysis, and generating downloadable PDF reports.

---

# 🚀 Features

✅ AI Molecule Generation  
✅ SMILES-Based Molecular Representation  
✅ Molecular Visualization using RDKit  
✅ Drug-Likeness Analysis  
✅ Toxicity Prediction  
✅ AI Confidence Scoring  
✅ Disease-Based Drug Discovery  
✅ Interactive Streamlit Dashboard  
✅ Downloadable PDF Reports  
✅ Modern Creative UI/UX Design
